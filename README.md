# [A702] Feeling filling
- 서비스 요약 
홧김비용을 홧김저금으로 만들어 드립니다.
우리 서비스는 사용자의 감정을 텍스트, 음성(STT)으로 입력받아
감정을 분석하고, 그에 맞는 기록과 저금 서비스를 제공합니다.
